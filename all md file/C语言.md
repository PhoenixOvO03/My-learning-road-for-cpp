PS：一定要自己动手敲，编程全程标点符号全部是英文，所有的分号逗号都是英文，中文的会报错，程序会运行不了，刚开始程序啥也不懂，你就照着抄就好了，后面都会慢慢理解的，刚开始理解不现实，一步一步来（加油，编程大牛第一步也是Hello World）

==程序==：
```c
#include <stdio.h>

int main()

}
```

==结果==：
```
echo:Hello World!
```
PS：上面的"echo:"是为了区分以后自己输入和电脑输出，实际输出的时候只有"Hello World!"

==踩过的坑==：
+ 全程所有标点符号都是英文的标点，中文的会报错
+ return后面的是0（zero），不是其他什么符号
+ 老的教学还写的void main，现在不这么写了（特殊情况除外）

==程序解析==：
`;`：分号是一个语句结束的标志，就像句号一样，表示一句话讲完了，是英文符号的";"不能打错了

`#include <stdio.h>`：这一句的意思是包含头文件，"std"是standard（标准）的缩写，后面的"io"表示的是input（输入）和output（输出），再后面的".h"是后缀名，表示是头文件，具体的作用我们后面会了解

`int main()`：整个程序的入口，有且仅有一个，具体的后面再分析

`{}`：整个程序就写在这个大括号里面，表示主函数的内容

`printf("Hello World!")`：用的是<stdio.h>中的printf()函数，用于打印输出里面双引号内的内容

`return 0`整个程序到这里就结束了，并返回一个值0，return具体的作用后面会了解

## 变量

我们编程，会定义变量，然后对变量进行操作，下面做个示范

==程序==：
```c
#include <stdio.h>

int main()
{
	// 定义了一个整数，名字叫num，初始值为10
    int num = 10;
    // 双引号内%d表示此处为一个整数，后面的num的值会在这里显示,\n表示输出换行
    printf("num = %d \n", num); // echo:10

	// num赋值为20
    num = 20;
    printf("num = %d \n", num); // echo:20

	// 定义一个num2，初始值为num(20)+10=30
	int num2 = num + 10;
	printf("num2 = %d \n", num2); // echo:30

    return 0;
}
```

PS：变量定义时，不能数字开头，不能和关键字相同（关键字由特殊意义，如：int），关键字不用特意去记，写程序会慢慢都知道的

==输出==：
```
echo:num = 10
echo:num = 20
echo:num2 = 30
```

==程序解析==：
定义变量的公式：
+ 变量类型  变量名;
+ 变量类型  变量名  =  初始值;

## 常量

常量就是不可变的量，比如我们会用到圆周率（3.14……），重力加速度（9.8），这些都是常量，不可修改，我们可以加一些修饰符，使得它不再可以更改

==程序==：
```c
#include <stdio.h>

int main()
{   
	// 定义一个常量PI，赋值为3.14
    const double PI = 3.14;
    // PI = 3.15; 假如执行该语句，是错误的，因为const修饰不可修改
    printf("PI = %lf \n", PI);// %lf为双精度浮点数

    return 0;
}
```

PS：const修饰的常量不可修改，如果修改，程序会报错

==输出==：
```
echo:PI = 3.140000 
```

# 运算符

## 赋值运算符

`=`：这个等号就是赋值运算符，会将等号右边的值复制一份粘贴给等号左边

==程序==：
```c
#include <stdio.h>

int main()
{   
    int a = 10;
    int b = a;
    a = 20;

    printf("a = %d , b = %d",a,b);

    return 0;
}
```

==输出==：
```
echo:a = 20 , b = 10
```

==程序解析==：
上出程序中b=a的操作并不是将a和b绑定起来，只是将a的值20，复制了一份给b，此时对a或者b操作，并不会对另一方造成影响

## 算数运算符

这里的算术运算符就是我们平时所用到的加减乘除了

| 运算符 | 含义 |
| ---- | ---- |
| + | 加 |
| - | 减 |
| * | 乘 |
| / | 除 |

==程序==：
```c
#include <stdio.h>

int main()
{   
    int a = 10;
    int b = 20;

	// 程序会将后面的a+b运算好后放到占位符
    printf("a + b = %d \n", a + b);
    printf("a - b = %d \n", a - b);
    printf("a * b = %d \n", a * b);
    printf("a / b = %d \n", a / b);
    printf("a / b = %lf \n", a * 1.0 / b);
    printf("a / b = %lf \n", (double)a / b);

    return 0;
}
```

输出：
```
echo:a + b = 30 
echo:a - b = -10      
echo:a * b = 200      
echo:a / b = 0        
echo:a / b = 0.500000 
echo:a / b = 0.500000 
```

==踩过的坑==：“^”这个不是乘方

==程序解析==：
可以看到程序最后三行行，因为a和b都是整数，那么，他们的运算结果默认也是整数，当a先和1.0做运算，那么此时a*1.0就是浮点数，此时和b做运算，结果也会是浮点数（隐式转换），先给a强制转换（显示转换）成浮点数，再和b运算，结果也为浮点数

隐式转换：当两个可以发生转换的类型进行运算，默认生成的结果为内存更大的那个类型

显示转换：把一个数据强制转换成你定义的类型

显示转换公式：(目标类型)变量名

## 分支结构

分支结构我们可以理解为数学里的分段函数，下面用`y = |x|`这个函数，我们可以得到有
+ y = x (x >= 0)
+ y = -x (x < 0)
我们依照这个函数，写出我们的第一个分支语句

==程序==：
```c
#include <stdio.h>

int main()
{   
    int x = -5;

    if (x >= 0) // 条件不满足
    {
        printf("y = %d",x);
    }
    else // if中的条件不满足
    {
        printf("y = %d",-x);
    }

    return 0;
}
```

==输出==：
```
echo:y = 5
```

==程序解析==：
分支结构可分为以下几种
+ `if(判断条件){程序体}`：当括号内的判断条件为true，执行程序体，否则离开分支结构，执行下面的语句
+ `if(判断条件){程序体1}else{程序体2}`：当括号条件为true，执行程序体1，否则执行程序体2
+ `if(判断条件1){程序体1}else if(判断条件2){程序体2}else{程序体3}`：判断条件1成立，执行程序体1，不成立则判断条件2，条件2成立执行程序体2，否则执行程序体3

分支可以创建很多很多个分支，当条件成立，则执行条件后的程序体，然后退出分支语句，如果都不满足，则执行else后面的程序体，如果没有else，则直接离开分支语句

PS：大括号{}包含的函数体为一个模块，当if满足会进入这个模块，不满足则退出这个模块，模块的概念不仅分支适用，在后续很多的知识都会用到这个概念

## 循环结构

当一个程序要反复执行一个语句时，要一遍一遍的写这个程序是不现实的，因为这个程序可能要执行成千上万遍，不可能写成千上万遍，所以有了循环结构，用来反复执行某一个代码

### while

==程序==：
```c
#include <stdio.h>

int main()
{   
    int num = 1;

    while (num < 5) // 括号里条件成立时，执行下面大括号内的程序
    {
        printf("%d ",num);
        /*
        以下几个等式相等
        num++;
        num += 1;
        num = num + 1;
        */
        num++;
    }

    return 0;
}
```

==程序输出==：
```
echo:1 2 3 4
```

### do-while

do-while和while的区别在于，do-while是执行后再判断，满足再执行，while是判断后再执行，观察两者的结构也能发现这个概念

==程序==：
```c
#include <stdio.h>

int main()
{   
    int num = 1;

    do // 先执行下面的语句
    {
        printf("%d ",num);
    } while (num > 10); // 不满足，离开循环体

    do // 先执行下方的语句
    {
        printf("%d ",num);
        num++;
    } while (num < 5); // 满足则继续循环，不满足离开循环
    
    return 0;
}
```

==输出==：
```
echo:1 1 2 3 4
```
