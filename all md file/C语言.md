# 梦的开始

## 第一个程序

具体上面的每个软件怎么配置软件、创建项目，就交给网络上的其他教程了，这里就不花时间过多赘述了（主要还是想一个md文件就写专门的一个方面的知识，这样条理清晰一些）

作为编程的老传统了，学啥都是先打印输出“Hello World”

PS：一定要自己动手敲，编程全程标点符号全部是英文，所有的分号逗号都是英文，中文的会报错，程序会运行不了，刚开始程序啥也不懂，你就照着抄就好了，后面都会慢慢理解的，刚开始理解不现实，一步一步来（加油，编程大牛第一步也是Hello World）

==程序==：
```c
#include <stdio.h>

int main()
{
    printf("Hello World!");

    return 0;
}
```

==结果==：
```
echo:Hello World!
```
PS：上面的"echo:"是为了区分以后自己输入和电脑输出，实际输出的时候只有"Hello World!"
恭喜你，就写出来了第一个C语言程序，离成为程序大牛又进了一步

==踩过的坑==：
+ 全程所有标点符号都是英文的标点，中文的会报错
+ return后面的是0（zero），不是其他什么符号
+ 老的教学还写的void main，现在不这么写了（特殊情况除外）

==程序解析==：
`;`：分号是一个语句结束的标志，就像句号一样，表示一句话讲完了，是英文符号的";"不能打错了

`#include <stdio.h>`：这一句的意思是包含头文件，"std"是standard（标准）的缩写，后面的"io"表示的是input（输入）和output（输出），再后面的".h"是后缀名，表示是头文件，具体的作用我们后面会了解

`int main()`：整个程序的入口，有且仅有一个，具体的后面再分析

`{}`：整个程序就写在这个大括号里面，表示主函数的内容

`printf("Hello World!")`：用的是<stdio.h>中的printf()函数，用于打印输出里面双引号内的内容

`return 0`整个程序到这里就结束了，并返回一个值0，return具体的作用后面会了解

## 注释

常言道：“我最讨厌两种人，一种是不写注释的，一种是让我写注释的”，程序在开发的时候是大家一起写的，你写的程序要让别人看懂，就要在上面写注解，也就是注释，主要是为了方便大家看懂代码（包括写代码多天后的自己）
常言又道：“这段代码今天只有我和上帝看得懂，明天就只有上帝看得懂了，再过几天，上帝也看不懂了”，所以，一定要养成写注释的好习惯
注释不会对程序有任何影响，更多的是写代码的一个小纸条，贴在那里，方便以后查阅

==程序==：
```c
// 一行注释

/*
一段注释
我也是哦
*/

int main()
{
	// 这是主函数内的一行注释
	
	/*
	这是主函数内的一段注释
	我也是哦
	*/

	return 0;
}
```

PS：后续的程序解析我几乎就写注释了

==程序解析==：
`//`：双斜线表示这一行后面的内容我都要写注释了（换行结束）
`/* */`："/*"后面都是注释，到 "\*/" 结束

## 变量

我们编程，会定义变量，然后对变量进行操作，下面做个示范

==程序==：
```c++
#include <stdio.h>

int main()
{
	// 定义了一个整数，名字叫num，初始值为10
    int num = 10;
    // 双引号内%d表示此处为一个整数，后面的num的值会在这里显示,\n表示输出换行
    printf("num = %d \n", num); // echo:10

	// num赋值为20
    num = 20;
    printf("num = %d \n", num); // echo:20

	// 定义一个num2，初始值为num(20)+10=30
	int num2 = num + 10;
	printf("num2 = %d \n", num2); // echo:30

    return 0;
}
```

PS：变量定义时，不能数字开头，不能和关键字相同（关键字由特殊意义，如：int），关键字不用特意去记，写程序会慢慢都知道的

==输出==：
```
echo:num = 10
echo:num = 20
echo:num2 = 30
```

==程序解析==：
定义变量的公式：
+ 变量类型  变量名;
+ 变量类型  变量名  =  初始值;

## 常量

常量就是不可变的量，比如我们会用到圆周率（3.14……），重力加速度（9.8），这些都是常量，不可修改，我们可以加一些修饰符，使得它不再可以更改

==程序==：
```c
#include <stdio.h>

int main()
{   
	// 定义一个常量PI，赋值为3.14
    const double PI = 3.14;
    // PI = 3.15; 假如执行该语句，是错误的，因为const修饰不可修改
    printf("PI = %lf \n", PI);// %lf为双精度浮点数

    return 0;
}
```

PS：const修饰的常量不可修改，如果修改，程序会报错

==输出==：
```
echo:PI = 3.140000 
```

## 占位符

在上面，我们有见过"%d"这种符号，我们称作占位符，表示在这里占了一个空间，由后面的数据替换，下面列一张表，表示一些常用的占位符

|占位符|含义|
|---|---|
|%d|以十进制形式输出整数|
|%u|以十进制形式输出无符号整数|
|%x|以十六进制形式输出整数（小写字母）|
|%X|以十六进制形式输出整数（大写字母）|
|%o|以八进制形式输出整数|
|%f|以浮点数形式输出实数|
|%e|以指数形式输出实数|
|%g|自动选择 %f 或 %e 输出实数|
|%c|输出单个字符|
|%s|输出字符串|
|%p|输出指针的地址|
|%n|输出已经输出的字符数|
|%%|输出百分号本身|

上面是占位符对应的含义，占位符还有他们的修饰符

|修饰符|含义|
|--|--|
|%-|左对齐输出|
|%0|在输出数字时，如果位数不够用 0 填充|
|%.|用于控制浮点数输出的小数位数|
|%*|用于控制输出的宽度，其中*表示要输出的宽度|
